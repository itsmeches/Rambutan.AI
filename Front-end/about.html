About RambutanAI

RambutanAI is an image-based AI system developed by Computer Science students as part of a deep learning course. It leverages computer vision to assess the ripeness and overall condition of rambutan fruits from images.

🎯 Our Mission

To support farmers, researchers, and agricultural stakeholders by providing an accessible tool for evaluating rambutan fruit health using AI-powered image classification.

🌿 Our Vision

To promote sustainable agriculture by offering smart technologies that improve decision-making, reduce post-harvest losses, and increase productivity—especially for local and small-scale farmers.

🧠 Technologies

  • 📚 Convolutional Neural Networks (CNN)
  • ⚙️ TensorFlow & Keras for deep learning
  • 🌐 HTML, CSS, JavaScript for the web interface
  • 🌐 Python flask for the backend logics
  • 📷 Custom image processing techniques
